Source Code

This project is open source. You can find the source code for the different parts of the project below.

Introduction

VisualKey is a complete ecosystem for creating, claiming, and empowering unique digital identities on EVM-compatible blockchains. It seamlessly combines a visual key generator with a dedicated utility token and a novel NFT collection. The entire project is open source, and the code for each component is available on GitHub.

Repositories

VisualKey UI

The Angular-based front-end application for the VisualKey project. It allows users to visually generate private keys and addresses, check balances, and interact with the NFT ecosystem. This is the main interface for the VisualKey platform.

  • VisualKey UI Repository

VisualKey API

Provides endpoints for Visual Keys NFT collection and token metadata, as well as on-the-fly image generation for NFT marketplaces. It serves metadata and images required by platforms like OpenSea.

  • VisualKey API Repository

Smart Contracts

The Solidity smart contracts for the VisualKey project. The contracts are ERC-721 and ERC-20 compliant and include advanced features like EIP-2612 permit, EIP-3156 flash loans, and signature-based minting.

  • Smart Contracts Repository

Rare Address Search (WASM)

A WebAssembly module built with Rust for generating and searching for rare cryptographic keys. This is used in the "Find Rare Keys" and "Vanity Address Generator" tools to perform computationally intensive tasks efficiently in the browser.

  • WASM Repository
