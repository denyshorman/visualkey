VisualKey Token

The Utility Token for the VisualKey Ecosystem

Introduction

VisualKey (VKEY) is the official utility token of the VisualKey ecosystem. It is the primary token required to use the platform's features, such as creating and enhancing Visual Key NFTs.

VKEY is designed with deflationary mechanics. When VKEY is used to mint a new NFT or increase its Power, those tokens are permanently removed from circulation (burned). This creates a direct link between platform activity and the total supply of VKEY.

VKEY is an ERC20 token with advanced features like flash loans and gas-less approvals for more efficient transactions.

Core Features

Controlled Supply Expansion

To support ecosystem growth, the VisualKey Token incorporates a mechanism for periodic supply increases, managed by the token owner. This allows for the introduction of new VKEY tokens in a transparent and predictable manner:

  • Approximately every 16 days, the token owner has the option to mint a new batch of 1,048,576 VKEY.
  • This minting opportunity is available for a 24-hour window following each 16-day interval. Unused minting opportunities do not accumulate.
  • The minting process is secured by a cryptographic signature from the owner to ensure authorized operations. This capability can also be permanently disabled by the token owner.

This controlled inflation model allows for strategic funding of new initiatives and community rewards.

Deflationary Mechanisms

The VisualKey Token includes several features that contribute to its scarcity over time:

  • Standard Burn Functionality: Token holders can permanently remove their VKEY from circulation using standard burn functions.
  • Flash Loan Fee Burning: The fee collected from EIP-3156 compliant flash loans is automatically burned, reducing the total supply with each transaction.
  • VKEY Burn on NFT Minting & Power-Ups: All VKEY tokens committed to mint a new Visual Key (VKEYNFT) or to increase an existing NFT's "Power" are permanently burned. This creates a direct link between the growth of the NFT ecosystem and the scarcity of VKEY.

These mechanisms ensure that activity within the VisualKey ecosystem contributes to a decreasing token supply.

Utility in the Visual Keys (VKEYNFT) Ecosystem

VKEY is integral to the Visual Keys NFT platform:

  • Visual Keys NFTs transform unique Ethereum addresses into verifiable digital collectibles, where the NFT's ID is the address itself.
  • The "Power" attribute of a VKEYNFT, a key rarity factor, is determined by the amount of VKEY committed (and subsequently burned) during its minting and any subsequent power-up events.
  • VKEY directly influences the distinction and potential desirability of each Visual Keys NFT.

Future Utility

VKEY is designed to be the primary token for future projects and services within the expanding digital landscape, ensuring its sustained relevance and utility.

Technical Standards

The VisualKey Token is built with a robust set of features for security, efficiency, and advanced interactions:

  • Gas-less Approvals (EIP-2612 Permit): Users can approve token spending via signatures, which can save on transaction fees and improve user experience.
  • Collateral-Free Loans (EIP-3156 Flash Loans): Allows for instant, uncollateralized borrowing of VKEY, provided the loan is repaid within the same transaction. The fee for such loans is configurable by the owner and is burned upon successful execution.
  • Enhanced Token Interactions (ERC1363): Supports `transferAndCall` and `approveAndCall` functionalities, allowing VKEY to interact with smart contracts in a single transaction.
  • Secure Ownership (ERC173): Implements a secure ownership model with a 16-day waiting period for ownership transfers of the token contract. The current owner can also cancel a pending transfer. The owner also has capabilities for emergency withdrawals of other tokens sent to the contract and for managing flash loan fees, all secured via signature.
  • Standardized Domain Separator (EIP-5267): Ensures compatibility and security for EIP-712 signature verifications.

Synergy with Visual Keys NFTs

VisualKey Token (VKEY) and Visual Keys NFTs are intrinsically linked to our VisualKey tool.

  • Address Generation: The VisualKey tool allows users to generate a unique Ethereum private key and its corresponding public address from a visual pattern.
  • Claiming an Address: A Visual Keys (VKEYNFT) represents one of these addresses as a permanent, collectible asset on the blockchain, where the NFT's ID is the address itself.
  • VKEY as a Catalyst: To mint a Visual Keys NFT and to later increase its "Power", VKEY is required. The VKEY tokens used in this process are then burned, contributing to the token's scarcity.

This synergy allows users to transform visual creations into verifiably scarce digital assets, with VKEY serving as the primary catalyst.
